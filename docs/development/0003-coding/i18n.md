# I18n

- **ALWAYS** wrap end user facing text in [Lingui macros](https://lingui.dev/ref/macro) `t` and `<Trans>`.
  - Prefer `<Trans>` to `t`.
- Use English for the source language in code.
- For admin website do not apply i18n, since it is used by internal users.

Some code examples are:

```tsx
import type { JSX } from 'react'
import { Trans } from '@lingui/react/macro'

export function MyComponent(): JSX.Element {
  return (
    <div className="flex-1 items-center justify-center gap-4">
      <Trans>Welcome!</Trans>
    </div>
  )
}
```

```tsx
import type { JSX } from 'react'
import { useLingui } from '@lingui/react/macro'
import { Stack } from 'expo-router'
import { ScreenAuthEmail } from '@/screens/auth-email'

export default function Screen(): JSX.Element {
  const { t } = useLingui()

  return (
    <>
      <Stack.Screen options={{ headerShown: false, title: t`Email` }} />
      <ScreenAuthEmail />
    </>
  )
}
```
