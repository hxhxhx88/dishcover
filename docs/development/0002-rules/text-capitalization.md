# Capitalization rule

Follow the [Google Developer Documentation Style Guide](https://developers.google.com/style/capitalization) for capitalization.

## Titles and headings

- Use **sentence case**: capitalize only the first word and proper nouns.
  - ✅ `## Capitalization in titles and headings`
  - ❌ `## Capitalization In Titles And Headings`
- Do not put a period at the end of a title or heading.

## UI elements and text

- Use **sentence case**:
  - ✅ `Sign up`
  - ❌ `Sign Up`
- **Colons**: start text after a colon with lowercase, unless it's a proper noun, heading, or quote.
  - ✅ `Note: the server is down.`
  - ✅ `Project: MyApp`
